## 🟣 3. 详细介绍（Full Description）

⭐ Quynj Translate Extension – 自由配置大模型的翻译插件

Quynj Translate 是一款轻量级、可自定义的大模型翻译工具。
用户可接入任意兼容 OpenAI API 格式的模型，实现高质量的单词解释与句子翻译。
支持多词性释义、多语言互译，是语言学习、写作与日常阅读的高效辅助工具。

## 🚗 快速体验

[Quynj Translate](https://quynj.github.io/translate-plugin)


## 🔧 主要功能

### 1. 自定义大模型 API（完全本地配置）
	•	自由填写 API 地址
	•	使用自己的 API Key
	•	可选择任意兼容 OpenAI API 的模型（如 Qwen、GPT 兼容模型、DeepSeek 兼容模型等）
	•	所有配置均 本地保存，不上传服务器

> 注意：模型需要response_format为json_object/json。模型能返回json类型的文本。

### 2. 单词翻译（多词性、多含义）

提供基于模型的更丰富解释，包括：
	•	不同词性分类
	•	多条语义释义
	•	更自然的语言说明
非常适合作文写作、英语学习、四六级、专升本英语等场景。

### 3. 句子与短文翻译（最多 1000 字符）

支持多语言互译，文本翻译流畅自然，可用于学习、阅读、写作辅助。

### 4. 温度调节（Precise ⇄ Creative）

可自由调整 Temperature：
	•	低温度：更精准、字面翻译
	•	中温度：更自然
	•	高温度：更具创造性

### 5. 可自定义 System Prompt

根据需要定义模型翻译风格，如“直译”“意译”“学术语气”等。
